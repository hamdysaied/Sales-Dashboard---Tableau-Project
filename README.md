# Sales Dashboard - Tableau Project

A comprehensive sales analytics dashboard built with Tableau Desktop Public Edition, providing interactive visualizations of sales performance across multiple dimensions.

## Overview

This interactive dashboard provides insights into sales data across subcategories, geographic regions, and time periods. It enables stakeholders to quickly identify trends, top-performing products, and regional sales distribution.

## Dashboard Components

### 1. Total Sales By Subcategory
- **Type**: Horizontal bar chart
- **Insight**: Phones ($330K) and Chairs ($328.4K) are the top-selling subcategories
- **Features**: Shows all 17 subcategories ranked by total sales value

### 2. Total Quantity By State
- **Type**: Interactive map visualization
- **Insight**: Geographic distribution of product quantities sold across US states
- **Features**: 
  - Bubble markers indicate quantity concentration
  - Hover functionality for detailed state-level data
  - Integrated Mapbox/OpenStreetMap layers

### 3. Total Profit By Region
- **Type**: Pie chart
- **Insight**: Regional profit distribution
  - West: $108.4K (largest segment)
  - East: $91.5K
  - South: $46.7K
  - Central: $39.7K

### 4. Total Sales By Year
- **Type**: Line chart
- **Insight**: Consistent upward sales trend from 2015 to 2018
- **Features**: Time-series visualization showing growth trajectory

### 5. Total Sales By Category & Subcategory
- **Type**: Treemap
- **Insight**: Hierarchical view of sales distribution
- **Categories**:
  - Technology (Phones, Machines, Accessories, Copiers)
  - Furniture (Chairs, Tables, Bookcases, Furnishings)
  - Office Supplies (Storage, Binders, Paper, Appliances)

## Key Metrics

- **Total Sales Range**: $3K (Fasteners) to $330K (Phones)
- **Time Period**: 2015-2018
- **Geographic Coverage**: United States
- **Product Categories**: 3 main categories, 17 subcategories

## Technical Details

- **Tool**: Tableau Desktop Public Edition
- **Data Source**: Sales transaction data
- **Visualizations**: 5 interactive charts
- **Map Integration**: Mapbox/OpenStreetMap

## Features

- Interactive filtering across all visualizations
- Cross-chart filtering capabilities
- Responsive design for different screen sizes
- Color-coded regional analysis
- Time-series trend analysis

## Use Cases

- Executive sales reporting
- Regional performance analysis
- Product category performance tracking
- Inventory planning based on sales patterns
- Strategic decision-making for sales and marketing teams

## Installation & Usage

1. Download and install [Tableau Desktop Public Edition](https://public.tableau.com/en-us/s/download)
2. Clone this repository
3. Open the `.twbx` file in Tableau Desktop
4. Interact with the dashboard using filters and hover functionality

This dashboard uses sample/anonymized data for demonstration purposes. Ensure compliance with data privacy regulations when using with real business data.
